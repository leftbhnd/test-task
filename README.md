# test-task

## Установка проекта
```
npm i
```


## Запуск бэка
```
node ./backend/server.js
```


## Запуск проекта
```
npm run serve
```


## Проверить код а стилистические ошибки
```
npm run lint
```


## Структура папок

+ `backend` - папка с бэкэндом
+ `src` - папка с проектом
+ `store` - файл `*.ts` содержащий управление и хранение данными
